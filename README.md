# Farid's Portfolio

## 🌐 Live Demo

https://faridhussain-portfolio.vercel.app

---

A modern, responsive portfolio built with **Next.js**, **TypeScript**, and **Tailwind CSS** to showcase my projects, technical skills, and experience as a Frontend Developer.

## ✨ Features

- Light & Dark mode with theme persistence
- Smooth scrolling powered by Lenis
- Subtle page and component animations using Framer Motion
- Active navigation highlighting while scrolling
- Project showcase with live demo and GitHub links
- Skills section with technology descriptions
- Functional contact form
- Email sending with Nodemailer
- Custom toast notifications
- Downloadable resume
- Client-side form validation
- SEO metadata and custom favicon
- Optimized images using Next.js Image component
